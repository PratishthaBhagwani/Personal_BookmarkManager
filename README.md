# Personal Bookmark Manager

A simple and beginner-friendly web application that allows users to **save, organize, and access their favorite websites** easily. This project focuses on clean UI, basic front-end logic, and usability.

---

## Features

* Display and organize bookmarked website links in a clean interface
* Clean and minimal user interface
* Beginner-friendly structure
* Easy to understand and extend

---

##  Tech Stack

* **HTML** – Structure of the application
* **CSS** – Styling and layout

---

##  Purpose of the Project

I created this project to:

* Practice **HTML & CSS fundamentals**
* Understand how simple web applications are structured
* Improve UI design and usability
* Build consistency on GitHub through hands-on practice

---

##  Future Improvements

* Add JavaScript for dynamic bookmark handling
* Implement delete/edit functionality
* Store bookmarks using browser local storage
* Improve responsiveness for mobile devices

---

##  How to Run

1. Clone the repository
2. Open `index.html` in a web browser
